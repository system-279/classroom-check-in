このシステムについて

Classroom Check-inは、オンライン講座の入室・退室を簡単に記録できるシステムです。 受講者がボタンを押すだけで、いつ講座に参加して、いつ退室したかを自動的に記録します。

主な特徴: ワンクリックで入退室を記録 / 滞在時間の自動計算 / 退室忘れの通知機能

使い方の流れ

1

ログイン

Googleアカウントでログインします

2

講座を選ぶ

参加する講座を一覧から選択します

3

INボタンを押す

入室ボタンを押して講座に参加します

4

講座を受ける

Classroomで講座を受講します

5

OUTボタンを押す

退室ボタンを押して完了です

よくある質問

OUTボタンを押し忘れたらどうなりますか？
システムが自動で通知を送信し、退室時刻の入力を促します。また、管理者が手動で退室時刻を設定することもできます。
複数の講座を同時に受講できますか？
いいえ、一度に参加できる講座は1つです。別の講座に参加する場合は、先に現在の講座を退室してください。
滞在時間はどのように計算されますか？
INボタンを押した時刻からOUTボタンを押した時刻までの時間が自動的に計算されます。

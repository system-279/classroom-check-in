このシステムについて

Classroom Check-inは、オンライン講座の入室・退室を簡単に記録できるシステムです。 受講者がボタンを押すだけで、いつ講座に参加して、いつ退室したかを自動的に記録します。

主な特徴: ワンクリックで入退室を記録 / 滞在時間の自動計算 / 退室忘れの通知機能

マルチテナント対応: 組織・学校ごとに独立したテナントを作成できます。データは完全に分離され、それぞれの組織で独自に管理できます。

よくある質問

テナントとは何ですか？
テナントは、組織や学校ごとに独立した環境のことです。テナントごとにデータが分離されているため、他の組織のデータを見ることはできません。 テナント登録後、専用のURLが発行されます（例: /your-org/admin）。
OUTボタンを押し忘れたらどうなりますか？
システムが自動で通知を送信し、退室時刻の入力を促します。また、管理者が手動で退室時刻を設定することもできます。
複数の講座を同時に受講できますか？
いいえ、一度に参加できる講座は1つです。別の講座に参加する場合は、先に現在の講座を退室してください。
滞在時間はどのように計算されますか？
INボタンを押した時刻からOUTボタンを押した時刻までの時間が自動的に計算されます。
受講者がログインできない場合はどうすればいいですか？
管理者がアクセス許可リストに受講者のメールアドレスを登録する必要があります。登録されていないメールアドレスではログインできません。

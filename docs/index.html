このシステムについて

Classroom Check-inは、オンライン講座の入室・退室を簡単に記録できるシステムです。 受講者がボタンを押すだけで、いつ講座に参加して、いつ退室したかを自動的に記録します。

主な特徴: ワンクリックで入退室を記録 / 滞在時間の自動計算 / 退室忘れの通知機能

マルチテナント対応: 組織・学校ごとに独立したテナントを作成できます。データは完全に分離され、それぞれの組織で独自に管理できます。

よくある質問

テナントとは何ですか？
テナントは、組織や学校ごとに独立した環境のことです。テナントごとにデータが分離されているため、他の組織のデータを見ることはできません。 テナント登録後、専用のURLが発行されます（例: /your-org/admin）。
OUTボタン（退室）が押せません
必要視聴時間に達していないことが原因です。 各講座には必要視聴時間（通常63分）が設定されており、この時間に達するまで退室できません。 画面上のプログレスバーと残り時間を確認してください。
OUTボタンを押し忘れたらどうなりますか？
システムが自動で通知メールを送信します。メール内のリンクからセルフチェックアウト機能を使って、ご自身で退室時刻を指定して打刻できます。管理者への連絡は不要です。
複数の講座を同時に受講できますか？
いいえ、できません。一度に参加できる講座は1つです。別の講座に参加する場合は、先に現在の講座を退室してください。
動画を倍速再生しても大丈夫ですか？
いいえ、倍速再生は禁止です。動画は通常速度（1倍速）で視聴してください。分割視聴も禁止です。最初から最後まで通して視聴してください。
必要視聴時間とは何ですか？
講座ごとに設定された「最低限視聴が必要な時間」です。デフォルトは63分（1時間3分）です。管理者が講座ごとに設定できます。この時間に達するまで退室ボタンは押せません。
滞在時間はどのように計算されますか？
INボタンを押した時刻からOUTボタンを押した時刻までの時間が自動的に計算されます。必要視聴時間に達しないと退室できないため、最低限の滞在時間が保証されます。
受講者がログインできない場合はどうすればいいですか？
管理者がアクセス許可リストに受講者のメールアドレスを登録する必要があります。登録されていないメールアドレスではログインできません。
